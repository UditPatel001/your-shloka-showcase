# Personal Portfolio

Welcome to my portfolio website.

This project is a reflection of my work, skills, and journey as a developer. It showcases the projects I've built, technologies I've worked with, and the ideas I'm currently exploring.

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* Vite
* shadcn/ui

## Running Locally

```bash
git clone <repository-url>
cd <project-folder>
npm install
npm run dev
```

The application will start on a local development server with hot reloading enabled.

## Features

* Responsive design
* Interactive user interface
* Project showcase section
* Skills and experience overview
* Contact and social links

I built this portfolio to create a central place for my projects, experiments, and achievements. It's a space that continues to evolve as I learn, build, and grow.
